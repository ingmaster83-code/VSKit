VSKit — VS Code 확장 프로그램 큐레이션

VSKit은 Visual Studio Code의 방대한 마켓플레이스에서 실제로 유용하고 검증된 확장 프로그램만을 엄선해 카테고리별로 정리한 큐레이션 서비스입니다. GitHub Copilot, GitLens, Prettier, ESLint 등 업계 표준으로 자리잡은 필수 확장부터 Error Lens, Thunder Client 같은 숨은 명작까지 소개합니다.

AI 코드 지원 카테고리에는 GitHub Copilot, Tabnine, IntelliCode 등 AI를 활용한 코드 자동완성 도구를 포함했습니다. 테마 카테고리에는 전 세계 개발자들이 가장 많이 사용하는 One Dark Pro, Dracula, Tokyo Night, Catppuccin을 소개합니다.

Git 카테고리의 GitLens는 VS Code에서 Git을 가장 강력하게 활용할 수 있게 해주는 확장으로, 인라인 blame, 코드 히스토리, 브랜치 비교 등 다양한 기능을 제공합니다. 언어 지원 카테고리에는 Python, Go, Rust, C/C++, Java, C# 등 주요 언어의 공식 확장을 포함했습니다.

모든 확장 프로그램은 VS Code 마켓플레이스 공식 링크를 통해 안전하게 설치할 수 있으며, 신뢰할 수 있는 개발사와 Microsoft 인증 확장을 우선적으로 수록했습니다.
