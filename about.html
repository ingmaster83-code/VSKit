💻 VSKit이란?

VSKit은 Visual Studio Code 마켓플레이스의 수천 개 확장 프로그램 중에서 실제로 유용하고 검증된 것만을 엄선해 카테고리별로 정리한 큐레이션 서비스입니다.

코드 지원·테마·Git·디버깅·언어·생산성·포매터 7개 카테고리로 분류된 필수 VS Code 확장을 한 곳에서 마켓플레이스 공식 링크로 연결해 드립니다.

✅ 서비스 특징

  • 직접 사용하고 검증한 확장 프로그램만 수록
  • VS Code 마켓플레이스 공식 링크를 통한 안전한 설치
  • 코드지원·테마·Git·디버깅·언어·생산성·포매터 7개 카테고리
  • Microsoft 인증 및 신뢰할 수 있는 개발사 확장 우선 수록
  • PC·태블릿·스마트폰 모든 기기에서 이용 가능
