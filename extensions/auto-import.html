확장 소개

Auto Import는 TypeScript·JavaScript에서 사용하는 모듈을 자동으로 import 구문으로 추가해주는 확장입니다. 함수나 클래스 이름을 입력하면 해당 모듈의 import를 파일 상단에 자동으로 추가합니다.

💡 Tips

  • 설치는 반드시 VS Code 공식 마켓플레이스에서만 받으세요.
  • Auto Import은(는) 무료입니다. 설치 후 VS Code를 재시작하면 즉시 사용할 수 있습니다.
  • Extensions ID: steoates.autoimport

자주 묻는 질문

Auto Import은 어떻게 설치하나요?

VS Code에서 Ctrl+Shift+X를 눌러 확장 탭을 열고 검색창에 이름을 입력하세요. 위 버튼을 눌러 마켓플레이스에서 직접 설치할 수도 있습니다.

Auto Import은 무료인가요?

무료입니다. 마켓플레이스에서 무료로 설치할 수 있습니다.

안전하게 설치할 수 있나요?

VS Code 공식 마켓플레이스에서 제공하는 링크로만 연결되어 있어 안전합니다. 항상 공식 마켓플레이스에서만 확장을 설치하세요.
